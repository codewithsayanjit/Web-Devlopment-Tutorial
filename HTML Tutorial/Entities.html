HTML Entities

HTML entities are special codes that represent characters in HTML. They are used to display reserved characters, special symbols, or characters that are not easily typed on a keyboard.

Common HTML Entities

  • & - Ampersand
  • < - Less than
  • > - Greater than
  •   - Non-breaking space
  • " - Double quotation mark
  • ' - Single quotation mark (apostrophe)
  • © - Copyright symbol
  • ® - Registered trademark symbol
  • ™ - Trademark symbol
  • ° - Degree symbol
  • $ - Dollar sign
  • € - Euro currency symbol
  • £ - Pound sterling symbol
  • ¥ - Yen currency symbol
  • ¢ - Cent currency symbol

Usage Example

To display the text "5 < 10" in HTML, you would write:

        <p>5 &lt; 10</p>

This will render as: 5 < 10
