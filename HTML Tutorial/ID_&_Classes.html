Understanding ID & Classes in HTML

What is an ID?

An ID is a unique identifier for an HTML element. It should be unique within a page, meaning no two elements should have the same ID.

<div id="uniqueElement">This is a unique element.</div>

What is a Class?

A class is a reusable identifier that can be applied to multiple HTML elements. Classes are used to group elements for styling or scripting purposes.

<div class="commonClass">This is a common element.</div>
<div class="commonClass">This is another common element.</div>

Using ID and Classes Together

You can use both IDs and classes on the same element to take advantage of their unique and reusable properties.

<div id="uniqueElement" class="commonClass">This element has both an ID and a class.</div>
