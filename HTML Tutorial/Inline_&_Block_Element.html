Understanding Inline and Block Elements

Block Elements

Block elements take up the full width available and start on a new line. Examples of block elements include:

  • <div>
  • <h1> to <h6>
  • <p>
  • <ul> and <ol>
  • <section>

Inline Elements

Inline elements do not start on a new line and only take up as much width as necessary. Examples of inline elements include:

  • <span>
  • <a>
  • <strong>
  • <em>
  • <img>
