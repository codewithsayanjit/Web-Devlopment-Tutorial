CSS Colors

This document provides an overview of how to use colors in CSS.

Color Names

You can use predefined color names in CSS:

  • Red
  • Blue
  • Green

Hexadecimal Colors

Colors can also be defined using hexadecimal values:

  • #FF5733
  • #33FF57
  • #3357FF

RGB Colors

RGB color values can be specified using the rgb() function:

  • rgb(255, 0, 0)
  • rgb(0, 255, 0)
  • rgb(228, 36, 199)

RGBA Colors

RGBA colors include an alpha channel for transparency:

  • rgba(255, 0, 0, 0.5)
  • rgba(0, 255, 0, 0.5)
  • rgba(0, 0, 255, 0.5)

HSL Colors

HSL colors are defined using hue, saturation, and lightness:

  • hsl(0, 100%, 50%)
  • hsl(120, 100%, 50%)
  • hsl(240, 100%, 50%)

HSLA Colors

HSLA colors include an alpha channel for transparency:

  • hsla(0, 100%, 50%, 0.5)
  • hsla(120, 100%, 50%, 0.5)
  • hsla(240, 100%, 50%, 0.5)
