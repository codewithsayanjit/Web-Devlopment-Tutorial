CSS Units Reference

Unit Description Example
px Pixels - absolute unit representing a single dot on the screen. width: 100px;
em Relative to the font-size of the element. font-size: 2em;
rem Relative to the font-size of the root element (html). margin: 1.5rem;
% Percentage - relative to the parent element's size. width: 50%;
vh Viewport height - 1% of the height of the viewport. height: 100vh;
vw Viewport width - 1% of the width of the viewport. width: 100vw;
cm Centimeters - absolute unit for physical measurements. width: 5cm;
in Inches - absolute unit for physical measurements. height: 2in;
