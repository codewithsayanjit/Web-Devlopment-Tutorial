# 🌐 Hosting Tutorial

A beginner-friendly guide to understand different types of web hosting and basic hosting-related commands.

---

## 📖 What is Hosting?

Hosting is a service that allows websites and web applications to be accessible on the internet. Hosting providers store your website files on servers and make them available to users worldwide.

---

# 🚀 Types of Hosting

## 1. Shared Hosting

### Description

Shared Hosting is a hosting environment where multiple websites share the same server resources such as CPU, RAM, and storage.

### Advantages

- Low cost
- Easy to manage
- Beginner friendly

### Disadvantages

- Limited resources
- Lower performance during high traffic
- Less control over server settings

### Best For

- Personal websites
- Blogs
- Small business websites

---

## 2. Dedicated Hosting

### Description

Dedicated Hosting provides an entire physical server exclusively for one user or organization.

### Advantages

- Maximum performance
- Full server control
- Enhanced security

### Disadvantages

- Expensive
- Requires technical knowledge

### Best For

- Large websites
- Enterprise applications
- High-traffic platforms

---

## 3. Managed Hosting

### Description

Managed Hosting is a hosting service where the hosting provider manages server maintenance, security updates, backups, and monitoring.

### Advantages

- Less maintenance
- Better security
- Automatic updates

### Disadvantages

- Higher cost than shared hosting
- Limited server customization

### Best For

- Businesses
- Developers
- WordPress websites

# 💻 Useful Hosting Commands

## Check Current Directory

```bash
pwd
```

## List Files and Folders

```bash
ls
```

## Create a New Directory

```bash
mkdir project-name
```

## Change Directory

```bash
cd project-name
```

## Remove Directory

```bash
rm -r project-name
```

## Copy Files

```bash
cp source.txt destination.txt
```

## Move Files

```bash
mv file.txt new-location/
```

# 🔧 Git Commands for Deployment

## Initialize Git Repository

```bash
git init
```

## Check Status

```bash
git status
```

## Add Files

```bash
git add .
```

## Commit Changes

```bash
git commit -m "Initial Commit"
```

## Connect GitHub Repository

```bash
git remote add origin <repository-url>
```

## Push Code

```bash
git push -u origin main
```

# 🎯 Conclusion

Choosing the right hosting depends on your website's requirements, budget, expected traffic, and technical expertise. Shared Hosting is suitable for beginners, Dedicated Hosting offers maximum control and performance, while Managed Hosting provides convenience and professional maintenance.
